# React Food Ordering Website

A React-based food ordering application that displays restaurants, allows users to search and filter restaurants, and view dynamic menu details using live API data.

## Features

- Restaurant listing page
- Search functionality
- Top Rated Restaurant filter
- Dynamic restaurant menu page
- Responsive user interface
- React Router for navigation
- Reusable React components
- Custom Hooks implementation
- Live API integration

## Tech Stack

- React.js
- JavaScript
- HTML5
- CSS3
- React Router

## Learning Outcomes

- Component-based architecture
- State management using React Hooks
- API fetching and rendering data
- Routing with React Router
- Building reusable UI components
